CakePHP 1.3

CakePHP is the version of CakePHP that follows 1.2. It has a slightly larger version number and more delicious goodness baked in. Although, not at a stable release, we've started documentation and migration guides for adventurous folk. Since 1.3 is a minor release, we've tried to make the upgrade path as painless as possible.

Migrating from 1.2 to 1.3

If you wish to upgrade an existing application from 1.2 to 1.3, then you should read the upgrade guide .

Delicious shiny new icing

If you just want to dive into the new things that have been added to 1.3 you should read The new features overview

I want to help with CakePHP 1.3. How can I do that?

Well there are a number of ways you can help.

  • The To-do list contains features we would like to implement. You can submit tickets and patches to remove items off the list.

  • Take it for a test drive. By giving things a try you can help find issues and open tickets to help speed up the resolution of any issues.
